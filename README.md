# Flask User Login System

A simple and secure **User Login System** built with **Python (Flask)**, **HTML**, **CSS**, **Bootstrap**, and **MySQL**.  
This project allows users to **register**, **log in**, and **update their profile information** securely.

---

##  Features

-  **User Registration** – Create a new user account with input validation.  
-  **Login Authentication** – Secure user login using Flask sessions.  
-  **Profile Update** – Users can edit and update their personal information.  
-  **Password Protection** – Passwords can be encrypted using hashing.  
-  **Responsive UI** – Built with Bootstrap for a clean, mobile-friendly design.  

---

##  Tech Stack

| Layer | Technology |
|--------|-------------|
| **Backend** | Python (Flask Framework) |
| **Frontend** | HTML, CSS, Bootstrap |
